# Changelog

All notable changes to the RTFact Jellyfin Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-11

### Added
- Initial release of RTFact Jellyfin Theme
- Inspired by Overseerr and ElegantFin designs
- Dark mode interface with purple-blue gradient accents
- Overseerr and ElegantFin-inspired color palette and design
- jsDelivr CDN integration for faster loading
- Inter font family integration
- Rounded cards and modern UI components
- Smooth transitions and hover effects
- Custom animated logo with gradient
- Dark gradient background for login page
- Responsive design for desktop and mobile
- Custom scrollbar styling
- Enhanced button styles with gradients
- Improved form input styling
- Media card hover effects
- Dashboard and settings page styling
- Context menu and tooltip styling
- Now playing bar customization
- Complete CSS variable system for easy customization
- Local preview HTML file for testing
- Comprehensive README with installation instructions
- MIT License

### Features
- **Color Scheme**: Deep dark backgrounds (#0f111a) with purple-blue accents (#5b5ee1, #8b5cf6)
- **Typography**: Inter font family for modern, clean text
- **Animations**: Smooth 0.25s transitions on interactive elements
- **Customization**: Easy-to-modify CSS variables
- **Compatibility**: Works with Jellyfin 10.11.2 and earlier versions
- **Performance**: Lightweight CSS-only theme with no JavaScript
- **CDN Delivery**: jsDelivr CDN for fast global access

### Design Elements
- 12px border radius for cards and components
- Gradient overlays on media items
- Elevated surfaces with subtle shadows
- Consistent spacing and padding
- Accessible color contrast ratios
- Modern layouts inspired by ElegantFin
- Improved animations and hover effects
